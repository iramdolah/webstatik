About Me

Hey there! Im Anaxa, a passionate beginner programmer on a quest to conquer the coding universe one line at a time. My journey into programming started with curiosity and a lot of YouTube tutorials, and now Im hooked on creating clean, efficient, and sometimes quirky code that actually works!.

Im currently diving deep into Python, my go-to language for everything from simple scripts to data manipulation and automation. Pythons straightforward syntax and versatility make it perfect for beginners like me, and I love how its widely used in web development, data science, and even AI! Im comfortable with core concepts like variables, loops, functions, and basic OOP, and Im constantly practicing to build small projects that solve real problems.

On the front-end side, Im exploring JavaScript, the language that brings websites to life. Ive learned how to manipulate the DOM, handle events, and create interactive web pages. JavaScripts flexibility fascinates me, and its thrilling to see how a few lines of code can transform a static page into a dynamic experience.

To take things further, Ive stepped into the world of React, where I get to build reusable components and manage state like a mini architect of web apps. Reacts component-based approach helps me organize my code better, making it easier to maintain and scale projects. Im learning about hooks, props, and conditional rendering, and Im excited about building responsive, user-friendly interfaces.

For the backend, Im working with Node.js, which lets me run JavaScript on the server side. This means I can now create complete web applications, from the user interface to the server logic. Im getting familiar with Express.js to handle routes and APIs, working with JSON data, and connecting to databases. Its amazing to build a full-stack app that talks to a database and serves data dynamically!

Even though Im still a beginner, Im dedicated to leveling up my skills every day by building projects, debugging like a detective, and reading documentation like its the newest best-seller. I believe programming is a superpower, and Im excited to keep learning new technologies and best practices.

When Im not coding, you can find me tinkering with new tools, watching programming tutorials, or getting distracted by memes about bugs and semicolons. Im all about sharing knowledge, collaborating with other coders, and contributing to open-source projects someday.

So, whether its writing Python scripts to automate boring tasks, creating interactive web pages with JavaScript and React, or building backend servers with Node.js, Im here to learn, grow, and make cool things happenone bug fix at a time!
